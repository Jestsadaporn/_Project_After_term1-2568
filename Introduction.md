# Project Watthana
###### จากโปรเจคการออกแบบและพัฒนาแอปพลิเคชัน watthana ที่ได้เคยพัฒนาไว้เมื่อ มัธยมศึกษาที่ 6 จากปัญหาการจดบันทึกเอกสารห้องเรียนสีขาวที่ถูกบันทึกอยู่ในรูปแบบกระดาษ ณ ตอนนั้นมีจุดประสงค์เพื่อต้องการเพิ่มประสิทธิภาพให้การบันทึกรายงานให้มีประโยชน์มากขึ้น สามรถร่วมใช้กับหน่วยงานต่าง ๆ ในโรงเรียนได้ และสะดวกต่อการบันทึกและแก้ไข ในเวลานั้นยังไม่ได้เรียนรู้เกี่ยวกับการพัฒนาด้าน UX /UI แต่หลังจากจากเรียนรู้ วิชา UX UI ของคณะ SIT KMUTT คนจบเทอมทำให้เกิดแรงบัลดาลใจนำเอาความรู้ที่ได้กลับมาพัฒนาแอปพลิเคชันนี้ และเพื่อเป็นประโยชน์ในการออกแบบแอปพลิเคชันเพื่อพัฒนาทักษะด้าน UX UI ต่อไป
