# วิเคราะห์หา Insight 
### :sunny: ในขั้นตอน Analytical Techniques 
#### Segmentation
###### แบ่งเป็น 2 กลุ่ม นักเรียน และ คุณครู 
###### จากข้อมูลที่นักเรียนให้มามีแพทเทินที่เหมือนๆกันคือ
###### 1.นักเรียนไม่เห็นถึงความสำคัญของการจดบันทึกข้อมูลห้องเรียนสีขาว (ความสำคัญ) เมื่อไม่เห็นถึงความสำคัญของการจดบันทึกทำให้เกิดการละเลย และต้องจดภายในระยะเวลาสุดท้าย 
###### 2. การจดในระยะเวลาสุดท้ายทำให้เกิดการปลอมข้อมูลขึ้นมา นักเรียนไม่ชอบตรงจุดนี้ (สิ่งที่นักเรียนไม่ชอบ) 
###### 3. การจดบันทึกข้อมูลห้องเรียนสีขาว มีขั้นตอนที่เยอะ นักเรียนไม่สามารถจดได้ทันทีหากสมุดบันทีกอยู่ที่อาจารย์ เพิ่มภาระให้นักเรียนที่ต้องไปรับมาจากอาจารย์ นอกจากจะต้องจดข้อมูลแล้ว (ภาระงาน) เมื่อจดบันทึกครบ ก็ต้องตามหาลายเซ็นจากอาจารย์ (ภาระงาน)
###### 4. นักเรียนชอบที่ระบบนี้สามารถที่จะจดบันทึกงาน/กิจกรรมในแต่ละวันได้ เป็นการเตือนความจำ 
###### 5. นักเรียนให้ความสะดวกสบายของระบบนี้เฉลี่ยอยู่ที่ 2.333
###### 6. นักเรียนมีความเห็นว่า ลดภาระของคนรับผิดชอบ ควรให้หัวหน้ารับผิดชอบเซ็นเอกสาร ควรตรวจสอบโดยหัวหน้าา / ควรทำให้อัปเดตได้เลยในวันนั้น ไม่ต้องตามย้อนหลังง ไม่เพิ่มภาระใคร  / ควรทำเป็นระบบออนไลน์มากกว่า เตรียมว่าควรเขียนวันไหน ควรมีการแต่งตั้งหน้าที่โดยไม่ผลักความรับผิดชอบไปให้ใคร
